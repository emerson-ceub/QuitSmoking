Quit Smoking 
O Quit Smoking e um programa que busca ajudar as pessoas a pararem de fumar, é possível colocar o dia em que alguém decidiu parar de fumar e visualizar isso em um painel que se atualiza sozinho com a quantidade de dia e horas que alguém parou de fumar, abaixo desse painel é possível visualizar os benefícios para a saúde em metas graduais. O site conta quantos cigarros alguém não fumou e o quanto dinheiro alguém economizou em consequência disso é o benefício para a saúde segundo a ciência. 

As informações são salvas usando LocalStorage do navegador de modo que as informações ficam persistentes na memória mesmo que o usuário feche a página.


Documentação
Para fazer o Download dos arquivos seguir os passos abaixo:
$ git clone https://github.com/emerson-ceub/QuitSmoking.git
cd ~/QuitSmoking no Linux/UNIX ou cd QuitSmoking no Windows.
firefox  QuitSmoking.html ou use o seu navegador preferido. 

Para usar o site basta colocar o dia e a hora em que você decidiu parar de fumar e apertar no botão “Começar Minha Jornada” e o contador se inicia imediatamente salvando os dados no LocalStorage, com isso você garante exatamente quantos cigarros deixou de fumar e os benefícios para a saúde e para o seu bolso.
